### week 10. 해시 / DFS & BFS

0-1. DFS 알고리즘이란 : 깊이우선탐색 알고리즘으로, 최대한 깊이 내려간 뒤, 더이상 깊이 갈 곳이 없을 경우 옆으로 이동한다.<br>
      자기자신을 계속해서 호출하는 순환 구조를 갖는 알고리즘
<br>

      자기자신을 계속해서 호출하는 순환 구조를 갖는 알고리즘
<br>
